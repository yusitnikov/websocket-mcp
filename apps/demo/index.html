How to Use

  1. Start the connection broker: connection-broker --port 3004
  2. Open this page in one or more browser tabs
  3. Start the MCP server: browser-mcp-server --broker ws://localhost:3004 --stdio
  4. Add the MCP server to Claude Desktop configuration
  5. Use Claude to control browser tabs via list_tabs and execute_js tools
